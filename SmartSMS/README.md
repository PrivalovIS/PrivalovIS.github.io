# SmartSMS

`SmartSMS` это локальное веб-приложение для отправки и приема SMS через GSM-модем по `Web Serial API`.

## Что есть сейчас

- интерфейс ближе к Telegram: слева компактный список чатов, контактов и вкладка настроек;
- статус модема всегда показан сверху окна;
- лог модема расположен под полем ввода и может скрываться;
- входящие SMS принимаются автоматически;
- чаты, контакты и настройки сохраняются в `IndexedDB`;
- в настройках есть импорт и экспорт базы данных;
- кириллица и латиница отправляются через `PDU/UCS2`.

## Запуск

Открывайте приложение в Chromium-браузере через `http://localhost` или `https://`.

Пример:

```powershell
npx serve .
```

После запуска откройте локальный адрес, подключите модем во вкладке `Настройки` и работайте с чатами.
