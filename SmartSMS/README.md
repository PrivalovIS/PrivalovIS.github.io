# GSM SMS Messenger

Локальное веб-приложение для отправки и приема SMS через GSM-модем, подключенный к COM-порту ПК, с использованием Web Serial API.

## Возможности

- интерфейс в формате мессенджера: список чатов слева, диалог справа;
- лог обмена с модемом внизу;
- отправка SMS через AT-команды;
- отображение входящих SMS;
- поддержка кириллицы и латиницы через UCS2;
- ручная синхронизация inbox и обработка уведомлений `+CMTI`.

## Запуск

Web Serial работает только в совместимых Chromium-браузерах и обычно требует `https://` или `http://localhost`.

Самый простой вариант:

1. Откройте папку проекта в локальном HTTP-сервере.
2. Запустите страницу через `http://localhost/...`
3. Нажмите `Подключить модем`.
4. Выберите COM-порт GSM-модема.

Пример запуска простого локального сервера, если установлен Node.js:

```powershell
npx serve .
```

После этого откройте адрес, который покажет сервер, обычно `http://localhost:3000`.

## Замечания

- Приложение настроено на текстовый режим SMS: `AT+CMGF=1`.
- Для Unicode используется `AT+CSCS="UCS2"` и `AT+CSMP=17,167,0,8`.
- Поведение некоторых модемов может отличаться. Если конкретная модель требует другие AT-параметры, их можно скорректировать в `app.js`.
