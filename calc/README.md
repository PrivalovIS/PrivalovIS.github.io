# Расчет связных номеров

Локальный офлайн-инструмент в одном файле `index.html` для расчета связных номеров по серийным номерам приборов учета.

## Как использовать

1. Откройте файл [index.html](C:/Users/priva/OneDrive/Рабочий%20стол/calc/index.html) в браузере.
2. Используйте вкладку `Одиночный расчёт` для расчета одного серийного номера.
3. Используйте вкладку `Массовый расчёт` для работы со списком серийных номеров.

Проект не требует установки зависимостей, сборки или запуска сервера. Файл работает локально.

## Возможности

- Одиночный расчет с мгновенным обновлением результатов.
- Массовый расчет в виде таблицы.
- Копирование отдельных результатов.
- Копирование всех строк или только связных номеров.
- Сортировка по серийному номеру и связному номеру.
- Подсветка ошибок, повторяющихся серийных номеров и одинаковых связных номеров.
